## Fő funkciók

Támogatja a proxyprofilok importját, megosztását és mentését, az egyérintéses kapcsolatot, késleltetéstesztet, böngészőalapú csomópont-diagnosztikát, DNS/FakeDNS javaslatokat, szabálycsoportokat, proxy láncokat és WebDAV mentést.

## Telepítés

Töltsd le az APK-t a GitHub Releases oldalról, majd nyisd meg Android eszközön. Ha az Android blokkolja, engedélyezd a telepítést böngészőből vagy fájlkezelőből. Nyisd meg az appot, engedélyezd a VPN-t, majd adj hozzá vagy importálj profilt.

## FAQ

A legtöbb telefonhoz arm64-v8a való. Az armeabi-v7a csak régi 32 bites eszközökhöz kell. Ha kapcsolódás után néhány webhely nem nyílik meg, futtasd a csomópont-diagnosztikát, és próbáld a helyi DNS/FakeDNS-t vagy a VPN DNS módosítását. Az új APK telepíthető a régi verzió fölé.
