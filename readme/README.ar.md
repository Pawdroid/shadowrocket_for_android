## الميزات

يدعم استيراد ومشاركة ونسخ ملفات الوكيل، الاتصال بلمسة واحدة، اختبار التأخير، تشخيص العقد عبر المتصفح، اقتراحات DNS/FakeDNS، مجموعات السياسات، سلاسل الوكيل، ونسخ WebDAV.

## التثبيت

نزّل APK من GitHub Releases وافتحه على جهاز Android. إذا منع النظام التثبيت، اسمح بالتثبيت من المتصفح أو مدير الملفات. افتح التطبيق، امنح إذن VPN، ثم أضف أو استورد ملفك.

## الأسئلة الشائعة

معظم الهواتف تستخدم arm64-v8a. استخدم armeabi-v7a فقط للأجهزة القديمة 32-bit. إذا لم تفتح بعض المواقع بعد الاتصال، استخدم تشخيص العقد وجرّب DNS المحلي/FakeDNS أو غيّر VPN DNS. يمكن تثبيت APK جديد فوق الإصدار القديم.
