## الميزات

يدعم استيراد ومشاركة ونسخ ملفات الوكيل، الاتصال بلمسة واحدة، اختبار التأخير، تشخيص العقد عبر المتصفح، اقتراحات DNS/FakeDNS، مجموعات السياسات وسلاسل البروكسي، ودعم WebDAV للنسخ الاحتياطي.

## التثبيت

نزّل APK من GitHub Releases وافتحه على جهاز Android. إذا منع النظام التثبيت، اسمح بالتثبيت من المتصفح أو مدير الملفات.

## الأسئلة الشائعة

معظم الهواتف تستخدم arm64-v8a. استخدم armeabi-v7a فقط للأجهزة القديمة 32-bit.
