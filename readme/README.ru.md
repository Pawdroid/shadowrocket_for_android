## Возможности

Поддерживаются импорт, отправка и резервное копирование proxy-профилей, подключение одним нажатием, тест задержки, диагностика узлов через браузер, советы DNS/FakeDNS, группы политик, proxy-цепочки и WebDAV-резервное копирование.

## Установка

Скачайте APK из GitHub Releases и откройте его на Android-устройстве. Если Android блокирует установку, разрешите установку из браузера или файлового менеджера. Запустите приложение, разрешите VPN и добавьте или импортируйте профиль.

## Вопросы

Для большинства телефонов подходит arm64-v8a. armeabi-v7a нужен только старым 32-битным устройствам. Если после подключения часть сайтов не открывается, запустите диагностику узла и попробуйте локальный DNS/FakeDNS или смените VPN DNS. Новый APK можно установить поверх старой версии.
