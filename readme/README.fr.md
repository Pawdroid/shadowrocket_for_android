## Fonctions principales

Prend en charge l’importation, le partage et la sauvegarde des profils proxy, la connexion en un geste, les tests de latence, le diagnostic de nœud via navigateur, les conseils DNS/FakeDNS, les groupes de stratégie, les chaînes proxy et la sauvegarde WebDAV.

## Installation

Téléchargez l’APK depuis GitHub Releases et ouvrez-le sur votre appareil Android. Si Android bloque l’installation, autorisez l’installation depuis le navigateur ou le gestionnaire de fichiers. Lancez l’app, accordez l’autorisation VPN, puis importez ou ajoutez un profil.

## FAQ

Pour la plupart des téléphones, choisissez arm64-v8a. Utilisez armeabi-v7a seulement sur les anciens appareils 32 bits. Si certains sites ne s’ouvrent pas après connexion, lancez le diagnostic de nœud et essayez le DNS local/FakeDNS ou changez le DNS VPN. Une nouvelle APK peut être installée par-dessus l’ancienne version.
