## Κύρια χαρακτηριστικά

Υποστηρίζει εισαγωγή, κοινή χρήση και αντίγραφα προφίλ proxy, σύνδεση με ένα πάτημα, έλεγχο καθυστέρησης, διάγνωση κόμβου μέσω browser, προτάσεις DNS/FakeDNS, ομάδες πολιτικής, αλυσίδες proxy και WebDAV backup.

## Εγκατάσταση

Κατεβάστε το APK από το GitHub Releases και ανοίξτε το στη συσκευή Android. Αν το Android μπλοκάρει την εγκατάσταση, επιτρέψτε εγκατάσταση από browser ή διαχειριστή αρχείων. Ανοίξτε την εφαρμογή, δώστε άδεια VPN και προσθέστε ή εισαγάγετε προφίλ.

## FAQ

Για τα περισσότερα τηλέφωνα χρησιμοποιήστε arm64-v8a. Το armeabi-v7a είναι μόνο για παλιές 32-bit συσκευές. Αν μετά τη σύνδεση δεν ανοίγουν κάποιες ιστοσελίδες, εκτελέστε διάγνωση κόμβου και δοκιμάστε τοπικό DNS/FakeDNS ή αλλαγή VPN DNS. Το νέο APK μπορεί να εγκατασταθεί πάνω από την παλιά έκδοση.
