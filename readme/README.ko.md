## 주요 기능

프록시 프로필 가져오기, 공유, 백업, 원터치 연결, 지연 시간 테스트, 브라우저 기반 노드 진단, DNS/FakeDNS 제안, 정책 그룹, 프록시 체인, WebDAV 백업을 지원합니다.

## 설치

GitHub Release에서 APK를 다운로드한 뒤 Android 기기에서 엽니다. 설치가 차단되면 브라우저 또는 파일 관리자 설치 권한을 허용하세요. 앱을 실행하고 VPN 권한을 승인한 뒤 프로필을 추가하거나 가져와 연결하세요.

## 자주 묻는 질문

대부분의 휴대폰은 arm64-v8a를 사용하면 됩니다. 오래된 32비트 기기만 armeabi-v7a를 사용하세요. 연결 후 일부 사이트가 열리지 않으면 앱의 노드 진단을 실행하고 로컬 DNS/FakeDNS 활성화 또는 VPN DNS 변경을 시도하세요. 새 APK는 기존 버전 위에 설치할 수 있습니다.
