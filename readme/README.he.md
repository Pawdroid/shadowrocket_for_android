## תכונות

תומך בייבוא, שיתוף וגיבוי פרופילי פרוקסי, חיבור בלחיצה אחת, בדיקת השהיה, אבחון צמתים דרך דפדפן, המלצות DNS/FakeDNS, קבוצות מדיניות, שרשראות פרוקסי וגיבוי WebDAV.

## התקנה

הורד את ה-APK מ-GitHub Releases ופתח אותו במכשיר Android. אם ההתקנה נחסמת, אפשר התקנה מהדפדפן או מנהל הקבצים. פתח את האפליקציה, אשר הרשאת VPN והוסף או ייבא פרופיל.

## שאלות נפוצות

ברוב הטלפונים יש להשתמש ב-arm64-v8a. armeabi-v7a מיועד רק למכשירי 32-bit ישנים. אם חלק מהאתרים לא נפתחים לאחר החיבור, הפעל אבחון צומת ונסה DNS מקומי/FakeDNS או שינוי VPN DNS. ניתן להתקין APK חדש מעל הגרסה הקיימת.
