## Mga tampok

May import, share at backup ng proxy profiles, one-tap connection, latency test, browser-based node diagnosis, DNS/FakeDNS suggestions, policy groups, proxy chains at WebDAV backup.

## Pag-install

I-download ang APK mula sa GitHub Releases at buksan sa Android device. Kung na-block ang installation, payagan ang install mula sa browser o file manager. Buksan ang app, payagan ang VPN permission, at mag-add o import ng profile.

## FAQ

Para sa karamihan ng phones, gamitin ang arm64-v8a. Gamitin lang ang armeabi-v7a sa lumang 32-bit devices. Kung may sites na hindi bumubukas matapos kumonekta, gamitin ang node diagnosis at subukan ang local DNS/FakeDNS o palitan ang VPN DNS. Maaaring i-install ang bagong APK sa ibabaw ng lumang version.
