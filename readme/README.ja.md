## 主な機能

プロキシプロファイルのインポート、共有、バックアップ、ワンタップ接続、遅延テスト、ブラウザベースのノード診断、DNS/FakeDNS の提案、ポリシーグループ、プロキシチェーン、WebDAV バックアップに対応しています。

## インストール

GitHub Release から APK をダウンロードし、Android 端末で開きます。インストールがブロックされた場合は、ブラウザまたはファイルマネージャーからのインストールを許可してください。起動後、VPN 権限を許可し、プロファイルを追加またはインポートして接続します。

## よくある質問

通常は arm64-v8a を選べば問題ありません。古い 32-bit 端末だけ armeabi-v7a を使用してください。接続後も一部サイトが開かない場合は、アプリ内のノード診断を実行し、ローカル DNS/FakeDNS の有効化または VPN DNS の変更を試してください。新しい APK は既存バージョンに上書きインストールできます。
