## Najważniejsze funkcje

Obsługuje import, udostępnianie i backup profili proxy, połączenie jednym dotknięciem, test opóźnienia, diagnostykę węzłów w przeglądarce, sugestie DNS/FakeDNS, grupy zasad, łańcuchy proxy i backup WebDAV.

## Instalacja

Pobierz APK z GitHub Releases i otwórz na urządzeniu Android. Jeśli Android blokuje instalację, zezwól na instalację z przeglądarki lub menedżera plików. Uruchom aplikację, udziel zgody VPN i dodaj lub zaimportuj profil.

## FAQ

Dla większości telefonów wybierz arm64-v8a. armeabi-v7a używaj tylko na starszych urządzeniach 32-bit. Jeśli po połączeniu część stron się nie otwiera, uruchom diagnostykę węzła i spróbuj lokalnego DNS/FakeDNS albo zmień VPN DNS. Nowy APK można zainstalować na starszej wersji.
