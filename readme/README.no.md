## Høydepunkter

Støtter import, deling og sikkerhetskopi av proxyprofiler, ett-trykks tilkobling, latenstest, browserbasert nodediagnose, DNS/FakeDNS-forslag, policygrupper, proxykjeder og WebDAV-backup.

## Installasjon

Last ned APK fra GitHub Releases og åpne den på Android-enheten. Hvis Android blokkerer installasjonen, tillat installasjon fra nettleser eller filbehandler. Åpne appen, gi VPN-tillatelse og legg til eller importer en profil.

## FAQ

De fleste telefoner bruker arm64-v8a. Bruk armeabi-v7a bare for eldre 32-bit-enheter. Hvis noen nettsteder ikke åpnes etter tilkobling, kjør nodediagnose og prøv lokal DNS/FakeDNS eller endre VPN DNS. Ny APK kan installeres over gammel versjon.
