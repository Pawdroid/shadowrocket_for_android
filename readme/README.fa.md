## ویژگی‌ها

از وارد کردن، اشتراک‌گذاری و پشتیبان‌گیری پروفایل پراکسی، اتصال یک‌لمسی، تست تأخیر، عیب‌یابی نود با مرورگر و پیشنهادات DNS/FakeDNS پشتیبانی می‌کند.

## نصب

APK را از GitHub Releases دانلود کرده و روی دستگاه باز کنید. در صورت مسدود شدن، امکان نصب از مرورگر یا فایل‌منیجر را فعال کنید.
