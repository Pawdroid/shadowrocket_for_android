## ویژگی‌ها

از وارد کردن، اشتراک‌گذاری و پشتیبان‌گیری پروفایل پراکسی، اتصال یک‌لمسی، تست تأخیر، عیب‌یابی نود با مرورگر، پیشنهاد DNS/FakeDNS، گروه سیاست، زنجیره پراکسی و WebDAV پشتیبانی می‌کند.

## نصب

APK را از GitHub Releases دانلود و روی دستگاه Android باز کنید. اگر نصب مسدود شد، نصب از مرورگر یا فایل‌منیجر را مجاز کنید. برنامه را باز کنید، اجازه VPN بدهید و پروفایل را اضافه یا وارد کنید.

## پرسش‌های متداول

برای بیشتر گوشی‌ها arm64-v8a مناسب است. armeabi-v7a فقط برای دستگاه‌های قدیمی 32-bit است. اگر بعد از اتصال بعضی سایت‌ها باز نشد، تشخیص نود را اجرا کنید و DNS محلی/FakeDNS یا تغییر VPN DNS را امتحان کنید. APK جدید روی نسخه قبلی نصب می‌شود.
