## Öne çıkanlar

Proxy profili içe aktarma, paylaşma ve yedekleme, tek dokunuşla bağlantı, gecikme testi, tarayıcı tabanlı node tanılama, DNS/FakeDNS önerileri, politika grupları, proxy zincirleri ve WebDAV yedeklemeyi destekler.

## Kurulum

APK’yı GitHub Releases sayfasından indirin ve Android cihazda açın. Android kurulumu engellerse tarayıcı veya dosya yöneticisinden kuruluma izin verin. Uygulamayı açın, VPN izni verin ve profil ekleyin ya da içe aktarın.

## SSS

Çoğu telefon için arm64-v8a kullanın. armeabi-v7a yalnızca eski 32-bit cihazlar içindir. Bağlandıktan sonra bazı siteler açılmazsa node tanılamayı çalıştırın ve yerel DNS/FakeDNS veya VPN DNS değiştirmeyi deneyin. Yeni APK eski sürümün üzerine kurulabilir.
