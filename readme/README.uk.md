# Shadowrocket for Android

**Сучасний VPN-клієнт для Android з керуванням proxy, тестуванням вузлів, маршрутизацією, резервним копіюванням і діагностикою мережі.**

## Основні можливості

Підтримує імпорт, поширення та резервне копіювання proxy-профілів, підключення одним дотиком, тест затримки і діагностику вузлів.
