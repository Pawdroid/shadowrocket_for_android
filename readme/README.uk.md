## Основні можливості

Підтримує імпорт, поширення й резервне копіювання proxy-профілів, підключення одним дотиком, тест затримки, діагностику вузлів у браузері, поради DNS/FakeDNS, групи політик, proxy-ланцюги та WebDAV backup.

## Встановлення

Завантажте APK з GitHub Releases і відкрийте на Android-пристрої. Якщо Android блокує встановлення, дозвольте інсталяцію з браузера або файлового менеджера. Відкрийте застосунок, надайте дозвіл VPN і додайте або імпортуйте профіль.

## FAQ

Для більшості телефонів підходить arm64-v8a. armeabi-v7a потрібен лише старим 32-бітним пристроям. Якщо після підключення деякі сайти не відкриваються, запустіть діагностику вузла і спробуйте локальний DNS/FakeDNS або змініть VPN DNS. Новий APK можна встановити поверх старої версії.
