## Höjdpunkter

Stöder import, delning och backup av proxyprofiler, anslutning med ett tryck, latenstest, browserbaserad noddiagnos, DNS/FakeDNS-förslag, policygrupper, proxykedjor och WebDAV-backup.

## Installation

Ladda ned APK från GitHub Releases och öppna den på Android-enheten. Om Android blockerar installationen, tillåt installation från webbläsare eller filhanterare. Öppna appen, ge VPN-tillstånd och lägg till eller importera en profil.

## FAQ

De flesta telefoner använder arm64-v8a. Använd bara armeabi-v7a för äldre 32-bitars enheter. Om vissa webbplatser inte öppnas efter anslutning, kör noddiagnos och prova lokal DNS/FakeDNS eller ändra VPN DNS. Ny APK kan installeras över gammal version.
