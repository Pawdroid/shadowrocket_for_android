## Recursos

Inclui importação, compartilhamento e backup de perfis proxy, conexão com um toque, teste de latência, diagnóstico de nós pelo navegador, sugestões de DNS/FakeDNS, grupos de política, cadeias proxy e backup WebDAV.

## Instalação

Baixe o APK no GitHub Releases e abra no seu dispositivo Android. Se o Android bloquear, permita a instalação pelo navegador ou gerenciador de arquivos. Abra o app, conceda permissão VPN e importe ou adicione um perfil.

## Perguntas frequentes

Para a maioria dos celulares, use arm64-v8a. Use armeabi-v7a apenas em aparelhos antigos de 32 bits. Se alguns sites não abrirem após conectar, execute o diagnóstico de nós e tente DNS local/FakeDNS ou altere o DNS da VPN. Uma nova APK pode ser instalada sobre a versão anterior.
