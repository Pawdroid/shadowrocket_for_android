## Ciri utama

Menyokong import, perkongsian dan sandaran profil proksi, sambungan sekali ketik, ujian latensi, diagnosis nod berasaskan pelayar, cadangan DNS/FakeDNS, kumpulan dasar, rantaian proksi dan sandaran WebDAV.

## Pemasangan

Muat turun APK daripada GitHub Releases dan buka pada peranti Android. Jika Android menyekat pemasangan, benarkan pemasangan daripada pelayar atau pengurus fail. Buka aplikasi, berikan kebenaran VPN dan tambah atau import profil.

## FAQ

Kebanyakan telefon menggunakan arm64-v8a. Gunakan armeabi-v7a hanya untuk peranti 32-bit lama. Jika sesetengah laman tidak terbuka selepas bersambung, jalankan diagnosis nod dan cuba DNS tempatan/FakeDNS atau tukar VPN DNS. APK baharu boleh dipasang di atas versi lama.
