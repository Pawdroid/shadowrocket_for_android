## मुख्य सुविधाएँ

Proxy profile import, share और backup, one-tap connection, latency test, browser-based node diagnosis, DNS/FakeDNS सुझाव, policy group, proxy chain और WebDAV backup समर्थित हैं।

## इंस्टॉल

GitHub Releases से APK डाउनलोड करें और Android device पर खोलें। अगर Android install रोकता है, तो browser या file manager से install की अनुमति दें। ऐप खोलें, VPN permission दें और profile add/import करें।

## FAQ

अधिकांश phones के लिए arm64-v8a सही है। armeabi-v7a केवल पुराने 32-bit devices के लिए है। connect होने के बाद कुछ websites न खुलें तो node diagnosis चलाएँ और local DNS/FakeDNS या VPN DNS बदलकर देखें। नया APK पुराने version पर install हो सकता है।
