## Funktionen

Unterstützt Import, Teilen und Backup von Proxy-Profilen, Ein-Klick-Verbindung, Latenztests, browserbasierte Node-Diagnose, DNS/FakeDNS-Empfehlungen, Richtliniengruppen, Proxy-Ketten und WebDAV-Backup.

## Installation

Lade die APK aus GitHub Releases herunter und öffne sie auf deinem Android-Gerät. Falls Android die Installation blockiert, erlaube Installationen aus Browser oder Dateimanager. Starte die App, erteile die VPN-Berechtigung und füge ein Profil hinzu oder importiere es.

## FAQ

Für die meisten Geräte ist arm64-v8a richtig. Verwende armeabi-v7a nur bei älteren 32-Bit-Geräten. Wenn nach dem Verbinden einige Websites nicht öffnen, nutze die Node-Diagnose und probiere lokales DNS/FakeDNS oder eine andere VPN-DNS-Adresse. Neue APKs können über die vorhandene Version installiert werden.
