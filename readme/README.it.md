## Funzioni principali

Supporta importazione, condivisione e backup dei profili proxy, connessione con un tocco, test di latenza, diagnosi dei nodi via browser, suggerimenti DNS/FakeDNS, gruppi di criteri, catene proxy e backup WebDAV.

## Installazione

Scarica l’APK da GitHub Releases e aprilo sul dispositivo Android. Se Android blocca l’installazione, consenti l’installazione dal browser o dal file manager. Apri l’app, concedi il permesso VPN e importa o aggiungi un profilo.

## FAQ

Per la maggior parte dei telefoni usa arm64-v8a. Usa armeabi-v7a solo su dispositivi vecchi a 32 bit. Se alcuni siti non si aprono dopo la connessione, esegui la diagnosi del nodo e prova DNS locale/FakeDNS o cambia il DNS VPN. Una nuova APK può essere installata sopra la versione precedente.
