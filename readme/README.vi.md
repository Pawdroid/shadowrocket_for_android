## Tính năng nổi bật

Hỗ trợ nhập, chia sẻ và sao lưu hồ sơ proxy, kết nối một chạm, kiểm tra độ trễ, chẩn đoán node bằng trình duyệt, gợi ý DNS/FakeDNS, nhóm chính sách, chuỗi proxy và sao lưu WebDAV.

## Cài đặt

Tải APK từ GitHub Releases rồi mở trên thiết bị Android. Nếu Android chặn cài đặt, hãy cho phép cài từ trình duyệt hoặc trình quản lý tệp. Mở ứng dụng, cấp quyền VPN, sau đó nhập hoặc thêm hồ sơ.

## Câu hỏi thường gặp

Hầu hết điện thoại nên dùng arm64-v8a. Chỉ dùng armeabi-v7a cho thiết bị 32-bit cũ. Nếu một số website không mở sau khi kết nối, hãy chạy chẩn đoán node và thử DNS cục bộ/FakeDNS hoặc đổi VPN DNS. APK mới có thể cài đè lên phiên bản cũ.
