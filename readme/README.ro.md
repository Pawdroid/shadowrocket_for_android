## Funcții principale

Acceptă importul, partajarea și backupul profilurilor proxy, conexiune cu o atingere, test de latență, diagnostic nod în browser, sugestii DNS/FakeDNS, grupuri de politici, lanțuri proxy și backup WebDAV.

## Instalare

Descărcați APK-ul din GitHub Releases și deschideți-l pe dispozitivul Android. Dacă Android blochează instalarea, permiteți instalarea din browser sau managerul de fișiere. Deschideți aplicația, acordați permisiunea VPN și adăugați sau importați profilul.

## FAQ

Pentru majoritatea telefoanelor folosiți arm64-v8a. armeabi-v7a este doar pentru dispozitive vechi pe 32 de biți. Dacă unele site-uri nu se deschid după conectare, rulați diagnosticul nodului și încercați DNS local/FakeDNS sau schimbați VPN DNS. APK-ul nou se poate instala peste versiunea veche.
